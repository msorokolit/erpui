ERP UI

Клієнтський ERP-інтерфейс

Цей застосунок динамічно завантажує data.txt, перетворює його на JSON-навігацію та форми, і рендерить їх на клієнті. Жодної серверної логіки.

  • Навігація формується із заголовків та відступів у data.txt
  • Поля форми визначаються мітками на кшталт ?R, ?N, ?S, ?D тощо
  • Збережені документи зберігаються у localStorage
Client-only ERP UI • data.txt → JSON • No backend
